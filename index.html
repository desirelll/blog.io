一、master 安装

master 节点所需 npm 包如下所示：

  • etcd-3.0.15-1.el7.x86_64.rpm

1. etcd 安装

执行 rpm -ivh etcd-3.0.15-1.el7.x86_64.rpm 安装 etcd，自动生成 /etc/etcd/etcd.conf 配置文件，配置文件通常不需要特殊的参数设置，默认将监听在 http://localhost:2379 地址。
安装完毕后通过 systemctl start 启动服务，同时通过 systemctl enable 将服务加入开机启动项中

通过执行 etcdctl cluster-health 检查验证是否成功启动

2. k8s-master 安装

修改 /etc/kubernetes/apiserver –insecure-bind-address=0.0.0.0

vue + webpack 部署到 Nginx

发表于 2018-04-16 | 分类于 frontend | | 阅读次数

一、打包项目

router mode 我选择了 hash，因为后端没有对 history 配置，这里要注意的是 base 要填对

打包之后生成的 dist 文件夹我放在了 /www/dist/，最好不要放在各种中英文加符号的一大串目录中，避免不必要的 bug 吧

Nginx 配置

注意这行 try_files $uri $uri/ /index.html;，因为我们是单页面项目，URL 中请求的地址实际是不存在对应映射的，所以将这些请求都转发到 index 中，不做这一步的话会出现刷新 404 的问题

启动

输入 sudo nginx 就可以启动了，访问 localhost:8080/dist 就可以

Queue & PriorityQueue

发表于 2018-02-27 | 分类于 java ， Java集合框架 | | 阅读次数

Queue 接口

Java 队列实现了元素先进先出的规则，即第一个添加的元素第一个被移除。

Queue 接口声明

Queue 的方法

方法 描述
boolean add(object) 插入指定的元素并返回true（插入成功）
boolean offer(object) 插入指定的元素
Object remove() 检索并移除队列第一个元素
Object poll() 检索并移除队列第一个元素，当队列为空时返回null
Object element() 检索第一个元素但不移除
Object peek() 检索第一个元素但不移除，当队列为空时返回null

PriorityQueue

PriorityQueue 为使用 queue 提供了便利，但不遵循先进先出原则，它继承自 AbstractQueue。
