kubernetes集群安装

集群环境：

  • master: 192.168.151.36
  • node: 192.168.151.51
  • node: 192.168.151.69

一、master 安装

master 节点所需 npm 包如下所示：

  • etcd-3.0.15-1.el7.x86_64.rpm
  • kubernetes-client-1.5.2-2.el7.x86_64.rpm
  • kubernetes-master-1.5.2-2.el7.x86_64.rpm

1. etcd 安装

执行 rpm -ivh etcd-3.0.15-1.el7.x86_64.rpm 安装 etcd，自动生成 /etc/etcd/etcd.conf 配置文件，配置文件通常不需要特殊的参数设置，默认将监听在 http://localhost:2379 地址。
安装完毕后通过 systemctl start 启动服务，同时通过 systemctl enable 将服务加入开机启动项中

通过执行 etcdctl cluster-health 检查验证是否成功启动

2. k8s-master 安装

执行 rpm -ivh kubernetes-client-1.5.2-2.el7.x86_64.rpm 和 rpm -ivh kubernetes-master-1.5.2-2.el7.x86_64.rpm，生成相关配置文件：

修改 /etc/kubernetes/config 配置中 –master 项为本机 IP 地址

修改 /etc/kubernetes/apiserver –insecure-bind-address=0.0.0.0
